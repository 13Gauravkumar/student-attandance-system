# Student Attendance Management System

## How to Run

**Option 1 — Standalone (no install)**  
Open `standalone.html` in your browser (double-click the file or use File → Open).

**Option 2 — Vite dev server**
```bash
npm install
npm run dev
```
Open [http://localhost:5173](http://localhost:5173).

---

## Tech Stack & Languages

- **Languages:** HTML, CSS, JavaScript
- **Standalone app:** Vanilla JS (no framework), single HTML file
- **Optional:** Vite, React (see `package.json`)
